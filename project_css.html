Emergency Response Beacon

An Emergency Response Beacon using IC 555 and IC 4017 is a simple flashing LED circuit for signaling in emergencies. The IC 555 generates clock pulses, which the IC 4017 decade counter uses to sequentially light up LEDs, creating a rotating or flashing effect. It is useful for rescue signals, roadside warnings, and security alerts.

Indoor Surveillance Bot

The Indoor Surveillance Bot is a smart robotic system built using an Arduino Mega and programmed in Embedded C via the Arduino IDE. It uses a PIR sensor for motion detection, an ultrasonic sensor for obstacle avoidance, a Bluetooth module for wireless control, stepper motors for movement, and a relay module to control external devices. This bot enhances indoor security by detecting intrusions, navigating spaces safely, and allowing remote operation. Measures temperature and humidity to monitor indoor climate in real-time.
